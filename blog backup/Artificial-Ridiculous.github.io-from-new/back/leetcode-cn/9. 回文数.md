# 9. 回文数

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

示例：  

### 输入1:   
`121`

### 输出1:  
```Python
True
```

### 输入2:   
`-121`

### 输出2:  
```Python
False
```

### 输入3:   
`10`

### 输出3:  
```Python
False
```


### 解答：  

```Python
def isPalindrome(self, x: int) -> bool:
    if x <0 or (x%10==0 and x!=0) :return False
    rev = 0
    while(x > rev):
        rev = rev*10 + x%10
        x //= 10
    return x == rev or x== rev//10
```
